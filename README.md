# Website-Replica

The project involved creating a website replica of Business Insider Website. Business Insider is a popular news website that covers a wide range of topics including business, financem tech, politics, and more. Replicating it onvolved designing and implementing a user interface that mirrors the layout and functionality of the Business Insider website.

Technologies used :
HTML
Used HTML to define the various sections of the Business Insider replica, such as header, main content area, sidebar, and footer.

CSS
To style HTML elements. Applied styles to replicate the visual appearances of Business Insider.

JavaScript
Implememted JavaScript to add interactivity to website. This includes features like sliders, dropdown menus, and any dynamic Content pressent on Business Insider Website.
